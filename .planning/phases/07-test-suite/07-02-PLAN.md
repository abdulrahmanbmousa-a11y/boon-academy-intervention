score_risk signature (src/risk_engine.py):
  def score_risk(df: pd.DataFrame) -> pd.DataFrame
  Adds columns: cfg.COL_RISK_SCORE, cfg.COL_RISK_LEVEL,
                cfg.COL_ATTENDANCE_RATE, cfg.COL_AVG_PRACTICE,
                cfg.COL_TREND_DIR, cfg.COL_DAYS_SINCE_NOTE
  Internal components: attendance_component (35%), practice_component (30%),
                       trend_component (20%), notes_component (15%).
  Weighted formula: 0.35*attendance_component + 0.30*practice_component +
                    0.20*trend_component + 0.15*notes_component = risk_score
  Risk level thresholds (cfg.RISK_THRESHOLD_CRITICAL=75, cfg.RISK_THRESHOLD_HIGH=50):
    score >= 75 -> "CRITICAL"
    50 <= score < 75 -> "HIGH"
    25 <= score < 50 -> "MEDIUM"
    score < 25 -> "LOW"
