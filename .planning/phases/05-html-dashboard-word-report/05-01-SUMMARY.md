## What Was Built

### src/config.py
- `DISPLAY_COLS_DASHBOARD: tuple[str, ...]` — 12 COL_* constants appended after `OUTPUT_COLS_CAMPUS`, following exact comment+style pattern.

### src/templates/dashboard.html.j2
Fully self-contained HTML file (no CDN, no external URLs):
- **CSS inline:** risk-level row colors (`#FFCCCC`, `#FFE5CC`, `#FFFFCC`, `#CCFFCC`) matching Phase 4 Excel palette; filter bar; expandable detail panel; stats bar.
- **JS inline:** `studentsData` const injected via `{{ students_json }}`; `filterAndRender()` reads campus filter, active risk button (`data-level`), and name search; `renderTable()` sorts by `risk_score` descending; row click toggles `<tr class="detail-row">` (one open at a time); `copyWhatsApp()` uses Clipboard API with `execCommand` fallback + 2s reset per D-06; `computeStats()` calculates total/critical/high/coverage% from `studentsData` on `DOMContentLoaded`.
- **Jinja2 variables:** `{{ students_json }}` (pre-serialised JSON string), `{% for cid in campus_ids %}` (campus filter options), `{{ run_timestamp }}` (footer).
